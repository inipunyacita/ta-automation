    2. Hasil testing dan scraping script yang dijalankan akan tampil, diantaranya :
    Hasil Test Basic Auth,
    Hasil Test Robot Tag,
    Hasil Test Footer Homepage,
    Hasil Test 404 Page,
    Hasil Test Footer Otherpage,
    Hasil Test Meta Tag
    Ketika script tidak mendeteksi tag meta (title & description), hasil akan menampilkan link dan informasi meta yang tidak ada

    Hasil Test Title & Link
    Ketika script mendeteksi link & tag title pada website, hasil akan menampilkan link dan informasi title untuk kemudian tester dapat cek kesesuaian link dengan title apakah identik/tidak

    Hasil Test Heading,
    Ketika script mendeteksi tag heading (h1,h2, dan h3), hasil akan menampilkan informasi urutan tag heading. Tester selanjutnya hanya perlu melihat urutan apakah sudah sesuai ketentuan tester checklist

    Hasil Test Alt Image
    Ketika script mendeteksi alt image yang tidak sesuai tester checklist, hasil akan menampilkan informasi link yang tidak sesuai.

    Hasil Test Breadcrumb
    Ketika script mendeteksi tag li pada tag ol yang digunakan untuk breadcrumb, hasil akan menampilkan informasi text dari tag li yang kemudian tester hanya perlu cek urutan apakah sudah sesuai dengan ketentuan tester checklist.

    3. Centang sesuai tester checklist pada spreadsheet :
    Link spreadsheet tester checklist dapat diakses melalui menu yang sudah terdapat pada halaman utama
